# Shortcuts

## Basic Navigation
* Orbit - Middle Click
* Pan - Shift + Middle Click
* Zoom - Scroll
* Top View - Numpad 7
* Front View - Numpad 1
* Side View - Numpad 3
* Camera View - Numpad 0
* Go to Selected Object - Numpad .
* Perspective/Orthographic - Numpad 5
* Zoom to Show All Objects - Home
* Fly Mode - Shift + F
* Toolbar - T
* Object Properties - N
* Reset 3D cursor - Shift + C
